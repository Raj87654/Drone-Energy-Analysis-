# Drone Battery Efficiency Analysis
# Project Overview

This project simulates drone delivery operations and analyzes how environmental and operational factors affect battery consumption.

# The main goal is to understand:

How distance impacts battery usage

Effect of wind and rain on energy consumption

Battery efficiency per kilometer

This project demonstrates real-world data analysis workflow including simulation, data cleaning, feature engineering, and visualization.

# Features of This Project

Synthetic data simulation (1000 deliveries)

Data corruption simulation (missing values, duplicates, invalid entries)

Data cleaning pipeline

Feature engineering

Visualization and analysis

# Variables Used

Distance_km – Delivery distance in kilometers

Package_weight_kg – Weight of the package

Wind_speed_mps – Wind speed in meters per second

Temperature_c – Ambient temperature in Celsius

Rain_intensity_mm – Rain intensity in millimeters

Battery_used_percent – Percentage of battery consumed per delivery

# Engineered Features

Energy_load – Combined impact of distance and package weight

Wind_impact – Influence of wind on battery consumption

Battery_per_km – Battery percentage used per kilometer
